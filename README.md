# Yui Amai's Blog

Personal academic blog built with [Astro](https://astro.build/) and based on the [Stellar](https://github.com/chrisgrieser/astro-theme-stellar) theme.

## Features

- Bilingual support (English/Japanese)
- Dark/light mode
- Responsive design
- Search functionality

## Development

```bash
npm install
npm run dev
```
