## Tailwind konfiguracija

Idite u `src/tailwind.config.mjs` da biste menjali promenljive.

Ne dirajte ništa unutar `screens` osim ako ne želite da menjate layout aplikacije.

`lightModeGradient` i `lightModeGradient` unutar
`backgroundImage` se koriste za background gradient koji možete da vidite na svakoj stranici.

Taj background gradient element se nalazi u base layout-u.

```html mark={3-5} title="src/layouts/Base.astro"
<body class="text-lightModeText dark:text-darkModeText">
  <nav />
  <div
    class="fixed left-0 top-0 z-[-2] h-screen w-full bg-lightModeBg bg-lightModeGradient transition-colors duration-[inherit] dark:bg-darkModeBg dark:bg-darkModeGradient"
  ></div>
  <div class="pt-20">
    <slot />
  </div>
</body>
```

## Stilizovanje kod blokova

U `src/astro.config.mjs` unutar `expressiveCode` objekta možete da menjate stilizovanje kod blokova.

```ts title="src/astro.config.mjs"
expressiveCode({
    themes: ['github-dark', 'github-light'],
    styleOverrides: {
    frames: {
        editorActiveTabIndicatorTopColor: 'transparent',
        editorActiveTabBorderColor: '#80808080',
        editorTabBarBorderBottomColor: '#80808080',
    },
    uiFontFamily: 'inherit',
    borderColor: '#80808080',
    },
}),

```

Ovde se nalazi cela expressive code [konfiguracija](https://expressive-code.com/reference/configuration/).

Unutar niza `themes` dodajte dve teme za blokove koda koje želite da koristite. Ovde se nalazi [lista](https://expressive-code.com/guides/themes/#available-themes) svih podržanih tema.

Onda te 2 teme dodajte u `src/config.ts` unutar `LIGHT_MODE_CODE_BLOCK_THEME` i `DARK_MODE_CODE_BLOCK_THEME`.

## Markdown stilizovanje

Za markdown stilizovanje ovaj projekat koristi plugin koji se zove [tailwindcss-typography](https://github.com/tailwindlabs/tailwindcss-typography)

Taj plugin pruža utility klasu zvanu `prose`, koja svim elementima daje neko stilizovanje.

Za skoro svaki element postoji prose utility klasa, npr. ako želite da promenite boju svih ul-ova u crno, samo napišite `prose-ul:text-black`

Zapamtite da ako želite da menjate stilizovanje nekih elementata, moraćete da dodate klasu i za tamni mod takodje.

Dodajte sve te klase unutar `src/layouts/Markdown.astro` u element sa `markdown` id-em.

## Ostalo stilizovanje

Unutar `src/styling` direktorijuma imate `code.css` i `scrollbar.css`.

<ul>
  <li>
    `code.css` se koristi za kod tagove u markdown fajlovima (ne za kod blokove)
  </li>
  <li>`scrollbar.css` se koristi za scrollbar stilizovanje</li>
</ul>
