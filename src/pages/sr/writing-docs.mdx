## Počnite

Prvo napravite markdown fajl (.md ili .mdx) i nazovite ga tako da odgovara <code>SidebarSchema</code> o kojoj smo ranije govorili.
Mora biti u kebab-case-u.

Hajde da napravimo markdown fajl koji se zove <code>get-started.mdx</code>

Unutar svakog markdown fajla morate staviti <code>title</code>, <code>description</code>, i <code>layout</code>.

U svaki markdown fajl stavite ovaj frontmatter na vrhu fajla:

```ts title="src/pages/sr/getting-started.mdx"
---
layout: '@/layouts/Markdown.astro'
title: Početak
description: Nauči kako da kreneš praviti svoju sledeću dokumentaciju sa Stellar-om.
---
```

<code>Layout</code> vrednost uvek mora biti ista.{' '}

The <code>title</code> i <code>description</code> bi trebalo
da budu prevedeni u isti jezik kao i jezik direktorijuma u kom se nalaze

npr.

Engleski prevod

```ts title="src/pages/en/getting-started.mdx"
---
layout: '@/layouts/Markdown.astro'
title: Getting Started
description: Learn how to get started with Stellar
---
```

Srpski prevod

```ts title="src/pages/sr/getting-started.mdx"
---
layout: '@/layouts/Markdown.astro'
title: Početak
description: Nauči kako da počneš korisiti Stellar.
---
```

Sad samo krenite da pišete ispod tog frontmatter-a

## Kod

Ovaj projekat koristi [expressive-code](https://expressive-code.com/) za kod blokove.

### Stilizovanje sintakse

Ako želite da dodate stilizovanje sintakse na blok koda dodajte kod jezika pored otvarajućih tikova:

<code>```ts</code>

const ime: string = 'Jovan'

<code>```</code>

To će izgledati ovako:

```ts
const ime: string = 'Jovan'
```

### Ime fajla kod bloka

Ako želite da dodate ime fajla na vrh kod bloka ukucajte title="ime-fajla.js" pored koda jezika kod otvarajućih tikova:

<code>```ts title="index.ts"</code>

const ime: string = 'Jovan'

<code>```</code>

To će izgledati ovako:

```ts title="index.ts"
const ime: string = 'Jovan'
```
