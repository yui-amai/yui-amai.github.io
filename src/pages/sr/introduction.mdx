## O projektu

Stellar je Astro.js tema za dokumentaciju.

Glavna svrha ovog projekta je da se autorima i održavaocima frameworka i biblioteka pruži šablon za dokumentaciju
koji mogu da koriste tako da se više fokusiraju na pisanje samih markdown fajlova nego na samo pravljenje i postavljanje projekta.

## Karakteristike

<ul>
  <li>Astro v4</li>
  <li>Tamni mod</li>
  <li>I18n sa typescript podrškom</li>
  <li>MD & MDX podrška</li>
  <li>Pretraga pomoću fuse.js</li>
  <li>Tailwind stilizovanje</li>
  <li>View transitions API</li>
  <li>Responzivnost & SEO tagovi</li>
  <li>Shiki stilizovanje sintakse koda</li>
  <li>Prilagodjenja putanja za importovanje</li>
</ul>

## Struktura projekta

```
├── public
├── src
│ ├── components
│ ├── i18n
│ │ ├── sidebar.ts
│ │ ├── ui.ts
│ ├── layouts
│ ├── pages
│ ├── styling
│ ├── config.ts
│ ├── types.ts
├── .eslintrc.cjs
├── .prettierignore
├── .prettierrc.cjs
├── astro.config.mjs
├── tailwind.config.mjs
├── tsconfig.json
```

<ul>
  <li>
    `public/` - Statičke stvari koje Astro neće obradjivati
  </li>
  <li>`components/` - Sve komponente koje se koriste u projektu</li>
  <li>
    `i18n/sidebar.ts` - U ovom fajlu je definisan `SIDEBAR` objekat koji služi za prikazivanje sidebar linkova u
    `src/components/sidebar/SidebarLinks.astro`
  </li>
  <li>
    `i18n/ui.ts` - Koristi se za definisanje svih jezika na kojim bi trebala 
    dokumentacija da bude prevedena kao i prevodi za korisnički interfejs
  </li>
  <li>`layouts/` - Svi layout-ovi koji se koriste u projektu</li>
  <li>
    `pages/` - Unutrar pages bi trebalo da bude `index.astro` (landing stranica za
    `DEFAULT_LANGUAGE` u `config.ts`) i `lang.astro` za svaki ostali jezik definisan u
    `LANGUAGES` u `src/i18n/ui.ts`

    Unutar pages ćete takodje staviti vaše markdown fajlove unutar svakog direktorijuma za svaki jezik.

  </li>
  <li>`styling/` - Koristi se za definisanje eksternih css fajlova</li>
  <li>`config.ts` - Koristi se za konfigurisanje celog projekta</li>
  <li>`types.ts` - Koristi se za definisanje često korištenih tipova</li>
  <li>`.eslintrc.cjs` - Eslint konfiguracioni fajl</li>
  <li>`.prettierignore` - U ovom fajlu stavite sve direktorijume i fajlove koje ne želite da budu formatirani i lintovani automatski</li>
  <li>`.prettierrc.cjs` - Prettier konfiguracioni fajl</li>
  <li>`astro.config.mjs` - Astro konfiguracioni fajl</li>
  <li>`tailwind.config.mjs` - Tailwind konfiguracioni fajl</li>
  <li>`tsconfig.json` - Typescript konfiguracioni fajl</li>
</ul>

## Komande

| Komanda                   | Šta radi                                                  |
| ------------------------- | --------------------------------------------------------- |
| `npm install`             | Instalira dependencije                                    |
| `npm run dev`             | Startuje lokalni dev server na `localhost:4321`           |
| `npm run build`           | Gradi ceo projekat unutrar `./dist/`                      |
| `npm run format`          | Formatira ceo projekat                                    |
| `npm run lint`            | Lintuje ceo projekat                                      |
| `npm run typecheck`       | Proverava da li ima typescript grešaka                    |
| `npm run preview`         | Pregleda vaš projekat lokalno, pre postavljanja           |
| `npm run astro ...`       | Pokreće CLI komande kao što su `astro add`, `astro check` |
| `npm run astro -- --help` | Potraži pomoć koristeći Astro CLI                         |
